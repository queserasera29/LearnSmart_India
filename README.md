# 📚 LearnSmart India — Digital Literacy Platform

## 🌟 Overview

**LearnSmart India** is a beginner-friendly digital literacy platform designed to educate users about essential internet skills, online safety, and digital financial awareness.

The platform provides **interactive learning modules in both English and Hindi**, followed by quizzes and a certificate upon successful completion.

This project focuses on **social impact**, helping users—especially beginners and non-English speakers—become confident and safe digital citizens.

---

## 🎯 Objectives

* Improve **digital literacy** among beginners
* Provide **bilingual education (English + Hindi)**
* Raise awareness about **online fraud and cybersecurity**
* Promote **safe digital payment practices (UPI)**
* Enable **self-paced learning with quizzes and certification**

---

## 🚀 Features

### 📖 Learning Modules

* Internet Basics
* UPI & Digital Payments
* Online Safety & Fraud
* Passwords & Privacy
* Social Media Awareness
* Digital Finance Basics
* Hindi Modules for accessibility

### 🧠 Interactive Quizzes

* Multiple-choice questions after each module
* Instant scoring system
* Minimum **60% required for certification**

### 🏆 Certificate Generation

* Auto-generated certificate on passing quiz
* Stores user data using **localStorage**
* Unique certificate ID for each learner

### 🌐 Bilingual Support

* English and Hindi modules
* Simplified language for wider accessibility

## ⚙️ How It Works

1. User selects a module from homepage
2. Lessons are displayed step-by-step
3. User attempts quiz
4. Score is calculated
5. If score ≥ 60%:

   * User enters name
   * Certificate is generated
6. Certificate data is stored locally
